## Basic Usage

### Initialize the Browser you want to use:

Syntax:
```
BROWSER="BROWSERNAME"
```

Currently supported browsers: "CHROME"

---

<br>

### Open a Website

Syntax:
```
OPEN "urltowebsite"
```

---

<br>

### Click on Elements

Syntax:
```
CLICK[index] CLASS/ID "class/id name"
```

Example:
```
CLICK[0] CLASS "readmore-btn"
```
